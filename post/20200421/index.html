git log の分析考える

21 April, 2020

（追記予定）

よく更新されるファイルを探したい

ブランチv1〜v2間の各ファイルの追加・削除行数の合計を表示し、追加行数降順に並べる

git log --numstat --no-merges --pretty="" v1 v2 | grep -v "dist/" | grep -E ".+\.(ts|tsx)" | sort -nr
